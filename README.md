# Library-Management
图书管理系统。下载即用，无需安装，使用软件所在路径作为保存文件位置。
中型小型大型图书馆，个人图书收藏家皆可使用。
大型图书馆多人图书管理员场景：软件暂不支持协同运作，可分划各自的图书管理区域，做到数据互不干扰。
